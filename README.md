# Interactive-Digital-Museum
---

### Project Description
**The Voices of History** is a real-time interactive digital museum powered by Gemini multimodal AI.
The system allows users to naturally converse with historical figures such as Tutankhamun, Nefertiti, Isaac Newton, and Leonardo da Vinci through voice, text, and images.
Users can speak to characters in real time, interrupt them naturally, and even show them photos of artifacts or historical objects. The AI analyzes the image and responds with contextual historical knowledge.
Each character appears as an animated guide inside a dynamic virtual museum environment, creating an immersive storytelling experience where history literally comes to life.
Key features include:
* Real-time multimodal conversation (voice, text, and images)
* Animated historical characters with synchronized speech
* Streaming AI responses with interruptible dialogue
* Vision-based artifact recognition
* Dynamic museum environments that adapt to the selected historical era

The project leverages Google Gemini multimodal models via the Google GenAI SDK and is deployed on Google Cloud Run within Google Cloud.
**The Voices of History** AI transforms museums from static spaces into living conversations with history.
